# DSA - Data Structures and Algorithms

This repository contains solutions to various LeetCode problems, organized by difficulty level. Each solution includes:
- Problem description
- Solution implementation in Python
- Time and space complexity analysis
- Test cases

## Repository Structure

```
DSA/
├── Easy/           # Easy difficulty problems
├── Medium/         # Medium difficulty problems
├── Hard/           # Hard difficulty problems
└── README.md       # This file
```

## Problems Solved

### Easy
1. **Two Sum** (#1) - Hash map approach for finding two numbers that sum to target
2. **Valid Parentheses** (#20) - Stack-based solution for bracket matching
3. **Reverse Linked List** (#206) - Both iterative and recursive approaches

### Medium
1. **Add Two Numbers** (#2) - Linked list digit addition with carry
2. **Longest Substring Without Repeating Characters** (#3) - Sliding window algorithm

## How to Run

Each problem solution can be run independently:

```bash
python Easy/two_sum.py
python Easy/valid_parentheses.py
python Easy/reverse_linked_list.py
python Medium/add_two_numbers.py
python Medium/longest_substring_without_repeating.py
```

## Testing

All solutions include test cases at the bottom of each file. Simply run the file to execute the tests.
